Tutorial page framework (How-to)

A tutorial should be a reliable step-by-step answer that supports both humans and AI.

  • Outcome (what users will achieve)
  • Prerequisites (machine/module/accessories/material)
  • Steps (numbered, concise)
  • Parameter table (starting ranges + array test)
  • FAQ (common failure modes)
  • Related (use case + troubleshooting + product links)
