What this prototype demonstrates

A single navigation + page framework that can live on toocaa.com (Shopify pages) while still linking to the existing Help Center, Blog, and Wiki. The goal is to make pages searchable (SEO), solvable (Support), and quotable (GEO / AI traffic).

SSOT rule
  • One canonical page for each topic (use <link rel="canonical">).
  • Other duplicates should 301 redirect or be set to noindex.
  • Keep a single “Link Pool” as the source of truth for URLs and mapping.
AI-ready structure
  • Use TL;DR, numbered steps, tables, and FAQ blocks.
  • Provide parameter cards and downloadable templates (jigs, fixtures).
  • Link to products and support pages as CTAs.
